# Tauri
Simple cheat for CS:GO.

Contains:
* Triggerbot,
* Glow,
* NoFlash,
* Bunnyhop.

Here is not implemented pattern scanner, all offsets are stored in output folder as Offsets.xml.
In case of game update, you need manually change values of those offsets.
